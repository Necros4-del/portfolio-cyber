Saé N°11

Se sensibiliser à l'hygiène informatique et à la cybersécurité

14h

Description du travail demandé

Vous devez réaliser un diaporama (~5 diapositives) qui présente les 4 bonnes pratiques qui vous semblent les plus importantes. Pour chacune d’elles, vous devez présenter le risque encouru, et la contre-mesure qui permet de minimiser ce risque.

Compétences développées

  • Analyse des risques : Identification des menaces liées à la cybersécurité et des bonnes pratiques adaptées.
  • Communication claire : Simplification de concepts techniques pour un public non spécialiste.
  • Création de supports : Production d'un diaporama structuré et attractif.
  • Sensibilisation et pédagogie : Transmission efficace de messages clés pour inciter au changement de comportement.
  • Recherche documentaire : Exploitation de ressources spécialisées pour enrichir le contenu.

Problèmes rencontrés

Volume d'informations élevé : Risque de surcharge d'informations dans un format limité (~5 diapositives).

Solution : Prioriser les pratiques les plus impactantes et pertinentes.

Grandes étapes

  • Recherches approfondit sur la sécurisation de mot de passe — source
  • Confection du diapositif sur Google Docs.

Saé N°22

Mesurer et caractériser un signal ou un système

Objectif & Contexte

Le professionnel R&T doit attester de la conformité des équipements après intervention en mesurant et caractérisant le système de transmission (NRO, NRA, ...).

Description du travail demandé

Appréhender, pour un système de transmission donné, les paramètres pertinents pour assurer une transmission de qualité, les mesures à effectuer, les appareils adéquats et leurs configurations. Analyser et présenter les résultats (cartographie radio, hypsogramme...).

Compétences / Problèmes / Solution

  • Analyser un signal fréquentiel (analyseur de spectre) ; utiliser et régler le matériel (balayage/FFT).
  • Interpréter des mesures, appliquer du filtrage numérique ; communiquer les résultats (diapo/soutenance).
  • Difficulté : maîtrise des outils de mesure. Solution : s’appuyer sur les cours et manuels en ligne.

Contenu des séances

  • Séance 1 : objectifs, organisation (7 séances, binômes), livrables.
  • Séance 2 : analyseur à balayage (GSP730) — réglages (Fc, Span, Ref level, RBW...).
  • Séance 3 : analyseur FFT — simulations Simulink (échantillonnage, Nyquist, aliasing...).
  • Séance 4 : filtrage numérique (filterDesigner, RIF, contraintes).
  • Séance 5 : point intermédiaire ; Séance 6 : mini-projet (RTL-SDR, spectre FM, filtre, démodulation) ; Séance 7 : soutenances.

Saé N°24 & Saé N°21

Projet intégratif

Description du travail demandé

Mettre en place l’infrastructure complète d’une entreprise hébergée dans une pépinière : réseau, services, téléphonie, sécurité et supervision. Solution évolutive, sécurisée et fonctionnelle (réseaux, Docker, virtualisation de services, scripts Python).

  • Conception réseau : plan clair (services, connexions internes/externes, point de sortie unique, redirection HTTP, nomenclature IP).
  • Planification : tâches, chronologie, organisation en « cellule IT ».
  • Configuration : switchs/routeurs (clients, serveurs, téléphones IP, caméras IP), DNS internes.
  • Clients & serveurs : parc Windows, Linux (Ubuntu), macOS ; 4 utilisateurs min., partages et profils par défaut.
  • Services Docker : TFTP, serveur Web multilingue (FR/EN) pages dynamiques, base de données.
  • Téléphonie IP : migration, QoS.
  • Supervision via LoRa : capteurs temp., affichage intranet, étude de portée.
